struct xmlURI

struct xmlURI {

    char *scheme;

    char *opaque;

    char *authority;

    char *server;

    char *user;

    int port;

    char *path;

    char *query;

    char *fragment;

};

A parsed URI reference. This is a struct containing the various fields as described in RFC 2396 but separated for further processing
